# MERN - unix

## Frontend:
##### Reactjs v18.2.0

## Backend:
##### NodeJS 


## Dependencies: 
Nodejs  v16.20.2
NPM     v8.19.4



-----------------------------------

## Start: 
#### Localhost

Start reactjs 

```bash
  cd/react
  npm i 
  npm run dev 
```

Start Nodejs  

```bash
  cd/node
  npm i 
  npm run dev 
```

## Deployment

Using docker

```bash
  docker-compose up -d 
```
